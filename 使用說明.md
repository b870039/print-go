# Print Go 使用說明

> 給工廠同仁的操作手冊，不需要懂程式也能上手。

---

## 系統在做什麼？

蝦皮有新訂單時，系統會**自動接單**，並檢查訂單有沒有問題。
你只需要打開瀏覽器，確認訂單、更新生產進度、標記出貨就好。

---

## 登入

用瀏覽器開啟系統網址，輸入帳號和密碼後按「登入」。

> 第一次使用請聯絡管理員取得帳號。

---

## 五個主要功能

### 1. 總覽（首頁）

登入後第一個畫面。

- 上方四個數字：全部訂單數、待確認、製作中、有問題
- 左下表格：最近進來的訂單
- 右下方塊：需要人工確認的問題訂單

有問題的訂單會顯示在這裡，**每天開始工作時先看這裡**。

---

### 2. 訂單管理

左側選單點「訂單管理」。

**篩選訂單**
畫面上方可以輸入關鍵字搜尋（訂單號、客戶名稱），或點選狀態分頁快速篩選。

**訂單狀態說明**

| 狀態 | 意思 |
|------|------|
| 待確認 | 剛進來的新訂單，尚未處理 |
| 有問題 | 系統發現疑問，需要人工確認 |
| 已確認 | 確認沒問題，可以開始製作 |
| 製作中 | 工廠已建立工單，正在生產 |
| 品檢 | 成品正在品質檢查 |
| 已出貨 | 已交給物流 |
| 完成 | 全部流程結束 |
| 取消 | 訂單已取消 |

**同步蝦皮訂單**
點右上角「🔄 同步蝦皮」可以手動把最新訂單拉進來（正常情況下系統會自動同步，這個按鈕是備用）。

**手動新增訂單**
非蝦皮來的訂單（電話、Line 接單）可以點「+ 新增訂單」手動建立。

---

### 3. 訂單詳情

在訂單列表點「詳情」可以看到這筆訂單的所有資訊：

- **基本資訊**：客戶、電話、備注、蝦皮單號
- **商品明細**：訂購了什麼、數量多少
- **盤查結果**：系統自動檢查的結果，有問題會列出來
- **狀態歷程**：這筆訂單每次狀態變更的紀錄
- **更新狀態**：最下方可以手動把訂單改成其他狀態

**建立工單**
訂單確認沒問題後，點右上角「建立工單」，這筆訂單就會出現在生產追蹤看板。

---

### 4. 問題盤查

左側選單點「問題盤查」。

這裡列出所有被系統標記為「有問題」的訂單，每筆訂單旁邊會說明是什麼問題，例如：

- **特殊需求**：客戶備注裡有「急」「客製」「特殊規格」等字，需要人工確認可不可以做
- **商品名稱缺失**：蝦皮訂單沒帶到商品名稱
- **缺少聯絡方式**：沒有電話也沒有地址

每筆訂單有兩個按鈕：
- **確認通過**：問題確認沒關係，繼續處理
- **取消訂單**：這筆訂單無法處理，直接取消

---

### 5. 生產追蹤

左側選單點「生產追蹤」。

**看板視圖**（預設）
從左到右是生產的每個階段：

```
排隊中 → 印刷中 → 後加工 → 品檢 → 包裝 → 待出貨
```

每張卡片是一筆工單，點擊卡片可以更新到下一個狀態。

**更新進度**
點任意工單卡片，選擇新的狀態，可以加備注，按「更新」。

出貨時記得填入**快遞追蹤號碼**，系統會自動把訂單標記為「已出貨」。

**列表視圖**
點右上角「列表」可切換成表格式，適合一次看很多工單。

---

## 常見問題

**Q：蝦皮新訂單沒有自動進來怎麼辦？**
到「訂單管理」頁面，點「🔄 同步蝦皮」手動拉取。如果還是沒有，請聯絡管理員確認蝦皮設定。

**Q：訂單狀態更新錯了怎麼辦？**
進入訂單詳情頁，在最下方「更新狀態」重新選擇正確狀態即可，系統會保留所有修改紀錄。

**Q：問題盤查的「AI標記」是什麼？**
系統的 AI 讀了客戶備注後認為有需要人工確認的地方，旁邊會附上 AI 的說明。看一下說明，判斷是否可以接單。

**Q：忘記密碼怎麼辦？**
請聯絡管理員重設密碼。

---

## 每日工作流程建議

1. **早上開機** → 看總覽，確認有沒有問題訂單
2. **問題盤查** → 逐一確認或取消問題訂單
3. **訂單管理** → 把「待確認」的訂單確認後建立工單
4. **生產追蹤** → 隨生產進度更新工單狀態
5. **出貨時** → 更新狀態為「已出貨」並填入快遞單號
